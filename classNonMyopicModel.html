Detailed Description

A model with a non-myopic optimal solution.

This model represents a world made of rooms. The rooms are divided in two equally sized groups.

The rooms in the first group are all interconnected, so that it is possible to step from any of them to any other with a single action.

The rooms in the second group are linearly connected, so that a single path exists that traverses all the rooms, one after the other. It is not possible to go back, so in each room only one choice is possible.

There is no way to go from a room in the first group to the second one.

The problem represented by the model is to find a single person, which is located inside a room (the state) and that moves randomly (outside of the agent control) at each timestep. The observer can only look, via its actions, inside the room to check if the person is there or not (so the size of the action space is the same as the state space: one action per possible room).

Note that this model returns no reward directly, as the problem's reward depends on the agent's belief (does it know where the person is with high accuracy?) and not on the current state of the world.
