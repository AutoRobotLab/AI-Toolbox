67 
68  // Since the original method of obtaining beliefs is stochastic,
69  // we keep trying for a while in case we don't find any new beliefs.
70  // However, for some problems (for example the Tiger problem) still
71  // this does not perform too well since the probability of finding
72  // a new belief (via action LISTEN) is pretty low.
