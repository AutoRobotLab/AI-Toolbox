Detailed Description

This class models a simple greedy policy.

This class always selects the greediest action with respect to the already obtained experience.

This function returns the probability of taking the specified action.

If multiple greedy actions exist, this function returns the correct probability of picking each one, since we return a random one with sampleAction().

Parameters
aThe selected action.
Returns
This function returns 0 if the action is not greedy, and 1/the number of greedy actions otherwise.

This function updates the greedy policy based on the result of the action.

We simply keep a rolling average for each action, which we update here. The ones with the best average are the ones which will be selected when sampling.

Parameters
aThe action taken.
rThe reward obtained.
