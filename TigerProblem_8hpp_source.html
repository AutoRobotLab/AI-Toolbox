6 // The following problem is the AAAI-94 Tiger problem, with
7 // a 0.95 discount factor. The problem can be stated as follows:
8 //
9 // The agent stands in front of two doors. He can open either of
10 // them. Behind one door, there is an agent-eater tiger, and
11 // in the other a small treasure. The agent does not know to what
12 // each door leads to, but instead of just opening the door, he
13 // can listen. When he listens, it will hear the tiger from either
14 // the left or right door. Its hearing is imperfect though, and
15 // 15% of the time it will hear the tiger behind the wrong door.
16 //
17 // Once the agent opens a door, it will either get a great penalty
18 // due to being eaten by the tiger, or get the reward. After that
19 // the game will automatically reset to an unknown state: the agent
20 // needs to start guessing again where the new tiger and treasure
21 // are.
22 //
23 // The states here are the positions of the tiger and treasure:
24 // since there are two doors, there are two states.
25 //
26 // There are three actions, corresponding to the listen action and
27 // open door actions.
28 //
29 // There are two possible observations, which are always random but
30 // for the listen action: in that case, we will obtain the correct
31 // information about the true state 85% of the time.
32 //
33 // The solutions of this problem have been computed using Tony
34 // Cassandra's pomdp-solve program (www.pomdp.org).
35
